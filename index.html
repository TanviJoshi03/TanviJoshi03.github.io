Featured Project

LLM Avalon Simulator Contributor

A multi-agent LLM simulator for The Resistance: Avalon, studying how language-model agents reason under hidden information, deception, and partial observability. Each agent receives a private role, participates in discussion, proposes teams, votes, and performs endgame assassination — all driven by LLMs. A rigorous testbed for emergent deceptive reasoning in language models.

Two-Tower Neural Recommendation System

Production-grade retrieval + ranking pipeline on MovieLens 1M, implementing the architecture used at YouTube and Pinterest. NDCG@10 +110% and MRR +85% with retrieve-then-rank over retrieval alone. FAISS ANN search for scalable candidate retrieval.

Domain-Specific RAG Pipeline with Evaluation

End-to-end Retrieval-Augmented Generation over 492 NSCLC PubMed abstracts. Compares chunking and retrieval strategies across Precision@5, MRR, and Faithfulness — treating evaluation as a first-class concern rather than an afterthought.

Hierarchical Time Series Forecasting

Forecasting retail sales across a 4-level hierarchy (Total → Department → Category → 30 Products). Compares ARIMA, Prophet, LSTM. Implements MinTrace-WLS reconciliation (Wickramasuriya et al. 2019) — recovering LSTM accuracy by 3.5% over unreconciled baselines and 31% over OLS MinTrace.

statsmodels Prophet PyTorch Time Series pandas

Persona Drift in Online Communities

Unsupervised detection of persona drift in Reddit communities using a Multi-View Drift Detector — fusing neural text embeddings with distributional baselines via weak supervision. No manual labels required. Built at UC San Diego (CSE 261).

NLP Reddit API Embeddings Weak Supervision

Customer Churn Prediction with Causal Attribution

Goes beyond prediction: explains why customers churn using SHAP-based causal attribution and K-Means cohort segmentation. XGBoost classifier with AUC-ROC 0.849 on IBM Telco dataset. Output: a targeted retention playbook.

Anomaly Detection on Time Series using GANs & VAEs

Generative model approaches to anomaly detection on multivariate time series data. Implements and compares GAN-based and VAE-based anomaly scoring methods — deep generative modelling applied to an unsupervised detection problem.
